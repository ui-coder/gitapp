/*  Git Commands
-------------------------------------*/
$ git status 
-> всегда пиши чтоб проверить состояние репозитория (что сейчас порисходит)

$ git add . 
-> эта команда добавляет все то новое что ты написал в сам Git. Это первая команда в списке тех которые нужны чтоб твои изменения появились в репозитории

$ git commit -m "Task name" 
-> Этой командой ты делаешь комит.

$ git push origin develop 
-> Этой командой ты закидываешь твой комит в сам репозиторий на GitHub
