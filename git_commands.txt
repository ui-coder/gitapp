/*  Git Commands
---------------------------------------
    Git Setup
-------------------------------------*/
$ git config --global user.email "you@email.com"
$ git config --global user.name "username"
$ git init - применяется только один раз для каждого нового репозитория
$ git clone [url]

$ git status 
-> всегда пиши чтоб проверить состояние репозитория (что сейчас порисходит)

$ git add . 
-> эта команда добавляет все то новое что ты написал(а) в сам Git. Это первая команда в списке тех которые нужны чтоб твои изменения появились в репозитории

$ git commit -m "Task name" 
-> Этой командой ты делаешь коммит.

$ git add . && git commit --amend --no-edit -> напиши эту строчку если коммит с названием таска уже делал(а) 

$ git push origin develop 
-> Этой командой ты закидываешь твой комит в сам репозиторий на GitHub
