Git Setup

Go back to Gitapp

$ git config --global user.email "your@email.com"

$ git config --global user.name "username"

$ git init — Применяется только один раз для каждого нового репозитория

$ git clone [url] — Копия репозитория на локальный копьютер

Git Workflow

$ git status — Всегда пиши, чтоб проверить состояние репозитория (что сейчас происходит)

$ git add . — Эта команда добавляет все то новое, что ты написал(а) в сам Git. Это первая команда в списке тех, которые нужны, чтоб твои изменения появились в репозитории

$ git commit -m "Task name" — Этой командой ты делаешь коммит

$ git push origin master/develop — Этой командой ты закидываешь твой коммит в сам репозиторий на GitHub
